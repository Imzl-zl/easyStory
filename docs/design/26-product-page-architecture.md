## 1. 页面总览

easyStory 包含以下页面：

| 序号 | 页面 | 路由族 | 功能定位 |
|---:|---|---|---|
| 1 | 认证 | `/auth/login`、`/auth/register` | 登录 / 注册 |
| 2 | 书架 | `/workspace/lobby` | 项目管理 + 公共配置 |
| 3 | 创作工作室 | `/workspace/project/[id]/studio` | 写作主场 + AI 对话 |
| 4 | 工作流引擎 | `/workspace/project/[id]/engine` | 自动化批量执行 + 工作流配置 |
| 5 | 分析实验室 | `/workspace/project/[id]/lab` | 研究外部文本 + 风格分析 |

## 2. 认证页

### 功能
- 登录
- 注册

---

## 3. 书架页

### 功能

**项目管理**
- 项目列表（卡片网格），每个卡片显示：项目名、目标字数、上次编辑时间、项目状态
- 卡片主操作：打开作品 / 继续创作
- 新建项目：可输入项目名或使用默认名称（如"新作品"），创建后直接进入工作室
- 回收站

**公共配置**（侧边面板）
- Skills
- Agents
- MCP
- 模型连接与凭证
- 账户信息

---

## 4. 创作工作室

### 功能

**文档树**（左侧栏）
- 按类别组织的项目文档：项目说明、设定、数据层、大纲（总大纲、开篇设计）、正文（章节列表）、时间轴、附录、校验
- 大纲、开篇设计、章节为系统文稿，不可重命名/删除
- 自定义文件/文件夹支持重命名、删除
- 新建文稿、新建目录、新建章节（工具栏 + 右键菜单均可触发）
- stale 章节标记、生成中章节加载动画

**编辑器**（中间区域）
- Markdown 编辑器（章节正文、大纲、开篇、设定）
- JSON 编辑器（数据层）
- 侧边栏可折叠，折叠后编辑器自动撑满
- 保存状态显示、字数统计

**AI 共创助手**（右侧栏，可折叠）
- 上下文感知对话（当前打开文档自动带入）
- AI 可读文档、写文档、搜索、生成大纲
- 返回内容可一键应用、追加或创建新文档
- Skill / 模型切换
- 写入权限控制

**质量可见性**（分阶段落地）
- 第 1 层：文档树标记（stale / 审核 issue 颜色标记）
- 第 2 层：审校面板（issue 列表、冲突详情、批量操作）
- 第 3 层：AI 聊天解释与修复

**提炼设定**
- AI 从正文、对话、文稿中自动提取项目设定，用户确认后更新项目说明和项目摘要

**进入工作流**
- 从工作室跳转引擎页，对当前项目/当前章节选择工作流执行自动化

**导出**
- 工具栏触发，选择范围/格式/版本后下载

## 5. 工作流引擎

### 功能

**执行管理**
- 选择工作流：内置工作流 / 项目工作流 / 最近使用
- 工作流启动、暂停、取消
- 执行历史列表
- 节点时间线（含状态、耗时、token 消耗）
- 实时执行进度

**节点详情**
- 输入/输出查看
- 执行日志
- Prompt 回放
- 审核结果查看
- 从节点结果跳回工作室查看/精修对应章节

**工作流配置**
- 内置工作流：系统提供默认创作流水线
- 项目工作流：用户可复制内置工作流后自定义节点、顺序、Skill、审核策略、导出节点
- 工作流管理仅在引擎页，不出现在书架或项目创建流程中
- 项目工作流归当前项目所有，用户可为不同作品维护不同工作流

**与工作室联动**
- 引擎生成的章节自动出现在工作室文档树
- 引擎执行中可切回工作室继续写作
- 工作室可主动跳转引擎对当前内容执行工作流

## 6. 分析实验室

> 当前为记录分析结果。上传外部文本自动分析、从结果生成 Skill 为后续目标能力。

### 功能

**分析记录**
- 创建分析记录，录入分析结果
- 筛选、查看、删除记录

**分析报告**
- 展示分析结果

**Skill 关联**
- 关联 Skill

## 7. 设置

设置入口分别在书架和工作室，可用侧边面板或独立设置页承载。

**个人设置**（书架页入口）
- 账户信息
- 模型连接（API key 等凭证在这里配，用户级）
- 个人规则
- Skills / Agents / Hooks / MCP

**项目设置**（工作室入口）
- 基础信息：项目名、目标字数、状态
- 项目摘要：查看 / 提炼 / 确认更新
- 项目规则
- Skills / MCP
- 审计记录
- 危险操作：删除 / 归档 / 清理

## 8. 功能覆盖矩阵

| 后端模块 | 承载位置 |
|---|---|
| `user` | 认证页 + 设置面板 |
| `project` | 书架页（列表/创建/删除/回收站）+ 工作室（文档树） |
| `content` | 工作室文档树 + 编辑器 |
| `workflow` | 工作流引擎页 |
| `review` | 引擎执行审核节点 + 工作室审校面板 |
| `context` | 工作室数据层 |
| `assistant` | 工作室 AI 共创助手 |
| `analysis` | 分析实验室 |
| `credential` | 设置面板 |
| `billing` | 引擎节点 token 消耗 |
| `export` | 工作室导出对话框 + 引擎导出节点 |
| `observability` | 引擎页日志/Prompt 回放 + 设置面板审计 |
| `config_registry` | 管理员独立入口（不暴露给普通用户） |

## 9. 空状态提示

| 位置 | 空状态引导 |
|---|---|
| 书架页（无项目） | 引导新建，提供输入框直接开始 |
| 工作室（无章节） | 引导在编辑器写第一章或在 AI 聊天对话 |
| 引擎页（无执行） | 引导选择或创建工作流 |
| 实验室（无分析） | 引导新建分析记录，后续支持上传参考文本 |
| 文档树数据层（空） | 提示 AI 可从正文自动提取 |
