## 1. 页面总览

easyStory 包含以下页面：

| 序号 | 页面 | 路由族 | 功能定位 |
|---:|---|---|---|
| 1 | 认证 | `/auth/login`、`/auth/register` | 登录 / 注册 |
| 2 | 书架 | `/workspace/lobby` | 项目管理 + 公共配置 |
| 3 | 创作工作室 | `/workspace/project/[id]/studio` | 写作主场 + AI 对话 |
| 4 | 工作流引擎 | `/workspace/project/[id]/engine` | 自动化批量执行 + 工作流配置 |
| 5 | 分析实验室 | `/workspace/project/[id]/lab` | 研究外部文本 + 风格分析 |

---

## 2. 认证页

### 功能
- 登录
- 注册

---

## 3. 书架页

### 功能

**项目管理**
- 项目列表（卡片网格），每个卡片显示：项目名、字数、上次编辑时间、工作流运行状态
- 新建项目：点击即创建，直接进入工作室，无中间孵化页
- 回收站

**公共配置**（侧边面板）
- Skills
- Agents
- MCP
- 模型连接与凭证
- 账户信息

---

## 4. 创作工作室

### 功能

**文档树**（左侧栏）
- 按类别组织的项目文档：项目说明、设定、数据层、大纲（总大纲、开篇设计）、正文（章节列表）、时间轴、附录、校验
- 大纲、开篇设计、章节为数据库真值源，不可重命名/删除
- 自定义文件/文件夹支持重命名、删除
- 右键菜单：打开编辑、导出、查看版本
- stale 章节标记、生成中章节加载动画

**编辑器**（中间区域）
- Markdown 编辑器（章节正文、大纲、开篇、设定）
- JSON 编辑器（数据层）
- 侧边栏可折叠，折叠后编辑器自动撑满
- 保存状态显示、字数统计

**AI 共创助手**（右侧栏，可折叠）
- 上下文感知对话（当前打开文档自动带入）
- AI 可读文档、写文档、搜索、生成大纲
- 返回内容可一键应用、追加或创建新文档
- Skill / 模型切换
- 写入权限控制

**质量可见性**（分阶段落地）
- 第 1 层：文档树标记（stale / 审核 issue 颜色标记）
- 第 2 层：审校面板（issue 列表、冲突详情、批量操作）
- 第 3 层：AI 聊天解释与修复

**其他**
- 导出：文档树右键或工具栏触发，选择范围/格式/版本后下载

---

## 5. 工作流引擎

### 功能

**执行管理**
- 工作流启动、暂停、取消
- 执行历史列表
- 节点时间线（含状态、耗时、token 消耗）

**节点详情**
- 输入/输出查看
- 执行日志
- Prompt 回放
- 审核结果查看

**实时事件流**
- SSE 推送执行进度

**工作流配置**
- 工作流模板管理（创建、编辑、删除）
- 模板仅在引擎页内管理，不出现在书架或项目创建流程中

**与工作室联动**
- 引擎生成的章节自动出现在工作室文档树
- 引擎执行中可切回工作室继续写作

---

## 6. 分析实验室

### 功能

**文本分析**
- 上传外部参考文本（小说等）
- 选择分析类型：风格分析、人物分析、结构分析等

**分析报告**
- 展示分析结果

**Skill 生成**
- 从分析结果提炼可复用 Skill
- 生成的 Skill 回到工作室可用

---

## 7. 设置

设置以侧边面板形式存在于书架和工作室中，不跳页。

**个人设置**（书架页入口）
- 账户信息
- 模型连接（API key 等凭证在这里配，用户级）
- 个人规则
- Skills / Agents / Hooks / MCP

**项目设置**（工作室入口）
- 基础信息
- 项目规则
- Skills / MCP
- 危险操作

---

## 8. 功能覆盖矩阵

| 后端模块 | 承载位置 |
|---|---|
| `user` | 认证页 + 设置面板 |
| `project` | 书架页（列表/创建/删除/回收站）+ 工作室（文档树） |
| `content` | 工作室文档树 + 编辑器 |
| `workflow` | 工作流引擎页 |
| `review` | 引擎执行审核节点 + 工作室审校面板 |
| `context` | 工作室数据层 |
| `assistant` | 工作室 AI 共创助手 |
| `analysis` | 分析实验室 |
| `credential` | 设置面板 |
| `billing` | 引擎节点 token 消耗 |
| `export` | 工作室导出对话框 + 引擎导出节点 |
| `observability` | 引擎页日志/Prompt 回放 + 设置面板审计 |
| `config_registry` | 管理员独立入口（不暴露给普通用户） |

---

## 9. 空状态提示

| 位置 | 空状态引导 |
|---|---|
| 书架页（无项目） | 引导新建，提供输入框直接开始 |
| 工作室（无章节） | 引导在编辑器写第一章或在 AI 聊天对话 |
| 引擎页（无执行） | 引导选择或创建工作流 |
| 实验室（无分析） | 引导上传参考文本 |
| 文档树数据层（空） | 提示 AI 可从正文自动提取 |
