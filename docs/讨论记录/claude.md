  5. 双工作模式

  模式设计：工作流级默认 + 节点级覆盖

  模式 1：手动模式（Manual Mode）
  workflow:
    mode: "manual"
    settings:
      auto_proceed: false  # 每个节点都需要人工确认
      auto_review: false   # 不自动审核

  用户操作流程：
  生成内容 → 暂停
    ↓
  用户操作：
    - 查看内容
    - 与 AI 对话修改（"把这段改得更激烈"）
    - 手动编辑
    - 手动触发审核（可选）
    - 确认通过
    ↓
  进入下一节点

  ---
  模式 2：自动模式（Auto Mode）
  workflow:
    mode: "auto"
    settings:
      auto_proceed: true   # 自动进入下一节点
      auto_review: true    # 自动审核
      auto_fix: true       # 审核失败自动精修
      max_retry: 3         # 最大重试次数

  执行流程：
  生成内容 → 自动审核 → 审核通过 → 自动进入下一节点
                    ↓
                审核失败 → 自动精修 → 重新审核（最多3次）
                                      ↓
                                  精修失败 → 暂停等待人工

  ---
  节点级覆盖：
  workflow:
    mode: "auto"  # 默认自动模式

  nodes:
    - id: "outline"
      auto_proceed: false  # 大纲节点强制人工确认

    - id: "chapter_1"
      # 继承 auto 模式，自动执行

  运行时切换：
  - 自动模式运行中 → 用户可以暂停，切换到手动模式
  - 手动模式下 → 用户可以选择"从这里开始自动执行"
