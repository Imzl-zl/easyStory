## 一、整体诊断

当前 UI 白皮书定义了 **4 个核心页面 + 2 个子视图**：

| 页面 | 覆盖功能 |
|------|---------|
| 灵感长廊 (Lobby) | 书架管理、开启新卷 |
| 创作孵化器 (Incubator) | 对话式设定 |
| 创作砚台 (Studio) | 编辑器、AI 对话 |
| 逻辑织网 (Engine) | 工作流编排、节点配置 |
| 文风实验室 (Lab) | 小说分析、风格提取 |
| 审核精修 (Review & Diff) | 对比视图 |

但功能需求（原设计 + 补充方案）定义了 **至少 15 个功能区域**，其中 **9 个没有对应的 UI 设计**。

## 二、缺失的页面/视图

### 🟡 10. 开篇设计 Wizard（前置资产确认向导）

前置资产实施计划要求"先想清楚再批量生成"。UI 需要一个引导式 Wizard，确保大纲和开篇设计在正文生成前得到确认。

#### 10.1 Wizard 流程设计

**标准创作链路**（对应后端 `outline -> opening_plan -> chapter_split -> chapter_gen`）：

```
┌─────────┐    ┌─────────┐    ┌─────────────┐    ┌──────────────┐
│ 项目设定 │ → │  大纲   │ → │  开篇设计   │ → │  章节任务   │
│ (Setting)│    │ (Outline)│   │(OpeningPlan)│    │(ChapterTask)│
└─────────┘    └─────────┘    └─────────────┘    └──────────────┘
    状态:        状态:          状态:             状态:
  未开始/草稿   待确认        待确认            待确认
  已确认/有变更 已确认        已确认            已确认
```

**前置条件**：新项目从零开始生成正文前，默认先确认 Outline 与 OpeningPlan。

#### 10.2 页面结构

**空间性格**：专注感，每一步聚焦单一任务，像翻阅一本启封的书。

**布局**：全屏模态向导，中心聚焦，四周留白。

- 保留手动进入向导的入口

### 🔴 1. 用户认证页面

补充方案 #4 要求多用户基础隔离。设计采用极简仪式感，像"展开宣纸准备创作"。

#### 1.1 登录/注册页面

**空间性格**：静谧感，大留白，无边框输入框，墨迹渗透过渡

**核心交互**：
- 登录/注册使用同一页面，通过 Tab 切换
- 切换时指示条滑动动画（300ms，spring curve）
- 输入框无边框，仅底部细线；focus 时渐变色彩从中间向两侧晕开
- 提交按钮悬停微缩 + 箭头位移

**视觉规格**：
- Logo 区域：64px 圆形图标，线条绘制动画（draw-in effect）
- 表单区域：白色卡片，1px 边框，`0 4px 8px rgba(0,0,0,0.02)` 阴影
- 提交按钮：`#171717` 背景，hover 变 `#2D2D2D`，scale(0.98)
- 加载状态：墨滴跳动动画

**动画规格**：
- Logo 圆圈：stroke-dashoffset 动画，1.5s ease-out
- Logo 十字：0.8s ease-out，delay 0.6s

- 输入框底线：width 0→100%，left 50%→0，300ms
- 按钮悬停：transform scale(0.98)，200ms
- 加载墨滴：scale 1→1.5，opacity 1→0.5，1s infinite

**简化设计**：邮箱输入 → 发送链接 → 显示提示（不做独立重置页面）

### 🔴 2. 全局设置抽屉 / 凭证管理中心

补充方案 #18 要求凭证管理（API Key 加密存储、连通性测试）。设计采用"折扇"从左侧拉出，敏感信息默认墨迹模糊。

#### 2.1 抽屉容器规格

**空间性格**：安全感的仪式感，功能分层清晰

**核心交互**：
- 抽屉从左侧拉出，420px 宽，100vh 高
- 背景叠加淡黑色遮罩 + 模糊效果
- Tab 切换，指示条滑动
- 关闭：点击遮罩或右上角关闭按钮

**动画规格**：
- 抽屉滑入：translateX(-100%) → translateX(0)，400ms，cubic-bezier(0.4, 0, 0.2, 1)
- 遮罩淡入：background rgba(0,0,0,0) → rgba(0,0,0,0.3)，blur 0 → 2px

### 7.1 视觉识别系统深度优化

#### Stale 状态的枯笔质感

**问题**：高分辨率显示器上，"淡墨色"文字可读性下降。

**优化方案**：引入"枯笔 (Dry Brush)"纹理。过时的章节文字不仅颜色变淡，且字体的边缘呈现出宣纸上水份不足的"飞白"效果。

| 要素 | 规格 |
|------|------|
| 文字色 | `#B5B5B5`（淡墨色） |
| 字体纹理 | `background-image: url("dry-brush-texture.svg")`，`background-clip: text` |
| 飞白效果 | 文字边缘 `text-shadow: 0 0 2px rgba(0,0,0,0.1)` 多层叠加 |
| 语义 | "失去水分/生命力"，视觉直观，不依赖对比度 |

#### 朱砂批注分级系统

**问题**：目前报错统称为"朱砂"，错误、警告、建议混淆。

**分级规格**：

| 等级 | 名称 | 色值 | 透明度 | 使用场景 |
|------|------|------|--------|----------|
| 致命 | 深朱 Deep Red | `#DC2626` | 实色填充 | Workflow Error、节点崩溃 |
| 警告 | 浅绛 Light Red | `#F87171` | 50% 晕染 | Stale 状态、Budget Warning |
| 建议 | 胭脂 Rose | `#FDA4AF` | 底部波浪线 | AI 润色建议、Prompt 优化提示 |

**视觉示例**：

```text
深朱 (致命): ████ 实色边缘 + 脉冲动画
浅绛 (警告): ░░░░ 半透明晕染 + 微弱呼吸
胭脂 (建议): ~~~~~~~~ 底部波浪线
```

### 7.2 Studio 交互微操优化

#### 上下文胶囊的 Translucent Vellum 效果

**问题**：多胶囊重叠时遮挡编辑器。

**优化方案**：胶囊背景采用"宣纸半透明"效果，参考不透明度。

| 要素 | 规格 |
|------|------|
| 背景 | `backdrop-filter: blur(12px)` + `background: rgba(255,255,255,0.8)` |
| 纸张纹理 | `background-image: url("vellum-texture.png")`，叠加 `5%` 透明度 |
| 边框 | `1px solid rgba(0,0,0,0.06)` |
| 重叠行为 | 底层文字"透光"隐约可见，不产生视觉阻塞 |

**ASCII 示例**：

```text
  ┌─────────────┐
  │ 大纲胶囊    │ ← 80% 毛玻璃
  └──────┬──────┘
         │ 重叠
  ┌──────▽──────┐
  │ 事实库胶囊   │ ← 底层内容透光可见
  │ ░░░林风░░░  │ ← 底层文字隐约透出
  └─────────────┘
```

#### 长篇小说的折扇式聚合

**问题**：500+ 章时，信封堆叠过长。

**优化方案**：按"卷"折叠为折扇形态。

```text
默认状态（只显示当前卷）:

  ┃ 第1卷 凡人修仙传 ▸ ┃    ← 折叠态
  ┃ 第2卷 仙界篇 ▸    ┃
  ┃ 第3卷 神界篇 ▸    ┃
  ┃ ···               │
  ┃ 第8卷 巅峰之战 ▸  ┃ ← 当前卷
  ┃                    │
  ┃ [📖 全部展开]      │

悬停展开态:

  ┃ 第1卷 凡人修仙传 ▾ ┃ ← 悬停展开
  ┃ ├─ 第1章 废物 (核心: 退婚)        │
  ┃ ├─ 第2章 神秘戒指 (核心: 药老)   │
  ┃ └─ 第3章 宗门大选               │
  ┃ 第2卷 仙界篇 ▸    ┃
```

### 7.3 Engine 生产力调优

#### Prompt Lab 浮动透视模式

**问题**：进入 Lab 后看不到当前节点在全局工作流中的位置。

**优化方案**：Lab 激活时，背景节点变暗，当前卡片放大并显示输入/输出连线。

```text
┌──────────────────────────────────────────────────────────────┐
│                                                              │
│      ┌─────────┐                     ┌─────────┐            │
│      │ 灵感提取 │                     │ 审核节点 │            │
│      └────┬────┘                     └────┬────┘            │
│           │ ╲   ← 透射的输入连线          ╱ │               │
│           │   ╲   (半透明虚线)         ╱   │               │
│           │     ╲                     ╱     │               │
│      ┌────┴──────▽──────────────────▽──────┴────┐          │
│      │          ██ Prompt Lab ██                  │ ← 放大  │
│      │          [当前节点: 章节生成]              │   110%  │
│      │                                        │          │
│      │    ╲ ╱ ← 透射的输出连线 (半透明)        │          │
│      └─────────────────────────────────────────┘          │
│                                                              │
│  其他节点: 墨云笼罩 opacity 0.3                            │
└──────────────────────────────────────────────────────────────┘
```

#### 模拟执行的墨滴动画

**问题**：点击模拟执行后，结果生硬弹出。

**优化方案**：结果以"水滴落入砚台"动画呈现，文字像墨迹一样慢慢析出。

| 阶段 | 动画 |
|------|------|
| 触发 | 卡片下方出现墨滴 |
| 滴落 | scale 1→0，opacity 1→0，300ms |
| 析出 | 文字逐字显现，20ms/字，ease-out |
| 完成 | 墨迹完全呈现，300ms 稳定 |

### 7.4 Lobby 作品感增强

#### 动态封面生成

**问题**：无封面时书架单调。

**优化方案**：系统根据题材自动生成水墨印章封面。

| 要素 | 规格 |
|------|------|
| 标题 | 宋体书法字体，居中 |
| 背景 | 基于题材的抽象墨迹（玄幻→山峦云雾，都市→高楼线条） |
| 印章 | 右下角圆形印章，内含分类标识 |
| 风格 | 水墨晕染一致性 |

#### 全局搜索 Ctrl+K 深度预览

**优化方案**：搜索结果显示伏笔和事实标签，点击定位到 Studio。

```text
按 Ctrl+K 后:

┌──────────────────────────────────────────────────┐
│ 🔍 搜索: 林风                                     │
├──────────────────────────────────────────────────┤
│                                                  │
│ 📖 第3章 宗门大选                                │
│    "...林风站在悬崖边..."                        │
│    🏷️ 伏笔: 戒指中的药老 (第1章埋下)            │
│                                                  │
│ 📖 第5章 异火觉醒                                │
│    "...林风手中的异火..."                        │

│                                                  │
│ 📖 第8章 家族复仇                                │

│                                                  │
```

**点击结果**：Studio 以"墨迹渗透"转场动画定位到对应行。

### 7.5 无障碍与舒适度

#### 暗色模式：玄青 + 月白

**优化方案**：黑色模式采用"玄青"底色，避免纯黑眩光。

| 模式 | 背景色 | 文字色 |
|------|--------|--------|
| 亮色 | `#FAFAFA` | `#171717` |
| 暗色 | `#0C0E14` (玄青) | `#E4E4E7` (月白) |

**效果**：减少长篇创作时的视觉残影。

#### 物理键盘反馈

**优化方案**：支持全键导航，确认操作支持 Cmd/Ctrl+Enter。

| 操作 | 快捷键 | 视觉反馈 |
|------|--------|----------|
| 确认下一步 | `Cmd/Ctrl + Enter` | 按钮产生"按压晕开"动画 |
| 取消 | `Esc` | 淡出 + 墨迹消散 |
| 焦点移动 | `Tab` / `Shift+Tab` | 焦点框渐变过渡 |

**按压晕开动画**：
```text
点击瞬间:
  scale(0.98) → scale(1.02) → scale(1)
  边缘扩散: box-shadow 墨迹晕开 0.3s
```

### 7.6 伏笔追踪 Scrollbar 墨点标记

**优化方案**：编辑器右侧 Scrollbar 旁显示伏笔墨滴，与伏笔抽屉互补。

```text
编辑器右侧:

  │█░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░│
  │█← 这是一条竖线（滚动条）          │|
  │        · ← 未干墨滴（未解决伏笔）  │
  │       ╱                           │
  │      · ← 闪烁（被提及但未回收）    │
  │     ╱                             │
  │    · ← 干燥墨渍（刚解决，将消失）  │
  │                                   │

状态说明:
  · 深色实心 = 未解决伏笔
  · 闪烁动画 = 伏笔在当前章节被提及
  · 淡出消失 = 伏笔已解决
```

## 五、交互流程总览（补充后）

```
注册/登录
  ↓
灵感长廊 (Lobby)
  ├── 开启新卷 → 创作孵化器 (Incubator) → 对话/模板/手填 → 创建项目
  ├── 继续执笔 → 创作砚台 (Studio)
  │     ├── 章节导航（左侧）
  │     ├── Markdown 编辑器（中间）
  │     ├── AI 对话助手（右侧/侧滑）
  │     ├── 上下文胶囊（顶部）→ 事实库查看
  │     └── 版本历史（侧面板）
  ├── 工作流 → 逻辑织网 (Engine)
  │     ├── 编排模式（静态配置）
  │     ├── 校验 → 错误列表展示
  │     ├── Dry-run → 成本预估确认
  │     ├── 执行模式（实时监控 + 进度 + Token 消耗）
  │     ├── 章节任务列表（大纲拆分后确认）
  │     └── 节点详情（上下文报告 + Prompt 回放 + 日志）
  ├── 文风分析 → 文风实验室 (Lab)
  │     ├── 上传 → 采样 → 分析
  │     ├── 分析报告（可编辑）
  │     ├── Skill 试写 → 保存
  │     └── Skill 作用域选择（全局/项目）
  ├── 导出 → 导出对话框（格式/范围/版本/排版）
  ├── 回收站 → 已删除项目（恢复/永久删除）
  └── 设置 → 账户/模型凭证/预算/系统
```
