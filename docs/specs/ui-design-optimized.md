# UI 设计文档（优化版）

## 概述

本文档定义 easyStory MVP 第一版的优化 UI 设计。相比之前的设计，本版本：
- 简化了导航结构
- 新增创作起点页面（自由对话式设定）
- 删除了 P1 功能（小说分析、快速开始、Skill 管理）
- 重构了项目详情页

**设计原则：**
- 简洁明了，避免过度设计
- 使用 shadcn/ui 组件库
- 响应式布局（桌面端优先）
- 清晰的信息层级
- 操作流程顺畅

**技术栈：**
- Next.js 16 + React 19.2
- Tailwind CSS 4
- shadcn/ui 组件库
- TipTap 富文本编辑器

## 设计规范

### 配色方案
- 使用 shadcn/ui 默认主题
- 主色：蓝色系（`primary`）
- 状态色：
  - 成功：绿色（`success`）
  - 等待：黄色（`warning`）
  - 错误：红色（`destructive`）
  - 进行中：蓝色（`info`）

### 间距规范
- 页面边距：`p-6`
- 卡片间距：`gap-4`
- 组件内边距：`p-4`
- 按钮间距：`gap-2`

### 字体规范
- 标题：`text-2xl font-bold`
- 副标题：`text-lg font-semibold`
- 正文：`text-base`
- 辅助文字：`text-sm text-muted-foreground`

### 响应式断点
- 桌面端：`lg:` (1024px+)
- 平板端：`md:` (768px+)
- 移动端：`sm:` (640px+)

### 交互原则
1. **操作反馈明确**：所有操作都有视觉反馈
2. **减少点击次数**：常用操作一键完成
3. **关键操作二次确认**：删除、拒绝等操作需确认
4. **加载状态提示**：异步操作显示加载状态
5. **错误提示友好**：错误信息清晰，提供解决方案
