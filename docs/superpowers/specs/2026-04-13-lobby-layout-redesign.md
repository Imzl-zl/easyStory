# 首页(Lobby)布局重设计 · 聚焦式书架

## 1. 问题分析

当前首页 `/workspace/lobby` 存在三个布局问题：

### 1.1 左侧边栏间距过大
- aside 容器 `p-4 lg:p-5`（16px/20px）+ 标题区 `px-1 pt-1` + 导航项 `p-3.5 pl-4`，累积约 38px（桌面端）顶部空白
- 标题区（"书架"小标签 + "作品空间"大标题）占据视觉重心但对导航无帮助

### 1.2 主区域信息层级混乱
- "继续创作"小字（L90）和"从书架回到故事现场"大标题（L91）是品牌文案，与用户核心任务（找作品/搜作品）无关
- 品牌标题区（~80px 高度）把搜索框推到视口下方，用户需要滚动才能看到

### 1.3 统计区信息密度不匹配
- 3 个独立 MetricCard 各约 120px，总计约 360px，但信息量仅 3 个数字
- "筛选结果"在无搜索时始终等于"当前作品"，信息冗余

## 2. 设计决策

### 2.1 侧栏：删除标题区，导航项前置
| 元素 | 当前 | 新设计 |
|------|------|--------|
| 标题区（"书架"/"作品空间"） | 存在，gap-2 px-1 pt-1 | **删除** |
| aside 容器 padding | `p-4 lg:p-5` | `p-3 lg:p-4` |
| 导航项 padding | `p-3.5 pl-4` | `p-2.5 pl-3` |
| 导航项 gap | `gap-1.5`（6px） | `gap-1`（4px） |
| "打开我的助手"按钮 | 主区域 header 右侧 | **移入侧栏底部** |
| "当前节奏"卡片 | 独立卡片（p-4 pb-5） | 精简为侧栏底部一行小字，格式：有模板时"当前节奏：模板名"，无模板时不显示 |

### 2.2 主区域：搜索框前置，统计精简
| 元素 | 当前 | 新设计 |
|------|------|--------|
| 品牌标题区（"继续创作" + 大标题） | header 独立区块，~80px | **删除** |
| 搜索框位置 | 第二个区块（section） | **第一个元素** |
| "新建作品"按钮 | header 右侧 | **搜索框同行右侧** |
| 统计区 | 3 个 MetricCard | **一行文字**：无搜索时"共 N 部作品"；有搜索时"筛选出 M 部作品" |
| 搜索框 padding | 包裹在 panel-shell 内 | 独立圆角输入框，无需外层 panel |

### 2.3 保持不变
- 顶部全局导航栏（WorkspaceShell）
- 项目卡片网格（LobbyProjectShelf）
- 侧栏导航项的数据和激活态逻辑（`LOBBY_NAV_ITEMS` 常量、`LobbyNavItem` 类型不变）
- "当前节奏"的信息来源（`templatePreviewNames`、`helperText` 变量保留）
- 搜索逻辑和状态管理（lobby-project-model）
- `projectCount` 和 `filteredProjectCount` 变量保留，用于统计文字

### 2.4 网格列宽调整
- `grid-template-columns` 从 `272px` 缩减到 `240px`（侧栏内容减少后宽度合理缩减）

## 5. 响应式适配

| 断点 | aside | main |
|------|-------|------|
| `< lg`（移动端） | aside 排序到下方（`max-lg:order-2`），导航项横向排列，"我的助手"导航项已覆盖助手入口 | 搜索框全宽 + 新建按钮独占一行 |
| `≥ lg`（桌面端） | aside 固定 240px（从 272px 缩减），sticky | 搜索框 + 新建按钮同行 |

> 移动端说明："打开我的助手"在桌面端移入 aside 底部；在移动端，aside 导航项中已有"我的助手"链接（横向排列可见），无需额外处理。

## 6. 视觉 Token 使用

所有改动复用现有 Design Token：
- 搜索框：`ink-input-roomy` 样式，独立使用不加 panel-shell 包裹
- 按钮：`ink-button`（新建作品）、`ink-button-secondary`（助手）
- 侧栏：`panel-glass` 保持
- 统计文字：`text-text-tertiary text-[0.84rem]`
- 助手入口：侧栏内虚线卡片 `border-dashed border-border` + `rounded-2xl`

## 7. 验收标准

- [ ] 左侧导航项顶部间距 ≤ 16px
- [ ] 搜索框为右侧主区域第一个可见交互元素
- [ ] "继续创作"小字和"从书架回到故事现场"大标题已删除
- [ ] "新建作品"按钮与搜索框同行
- [ ] 统计区为一行文字（无搜索时"共 N 部作品"，有搜索时"筛选出 M 部作品"），无 MetricCard
- [ ] "打开我的助手"桌面端移入侧栏底部，移动端通过导航项"我的助手"可见
- [ ] 项目卡片网格布局和功能不变
- [ ] 移动端响应式正常（aside 下移、导航横向）
